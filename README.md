## 简介
文件对比工具BreadcrumbsAraxis-Merge-Pro的语言dll，因为net上没找到的2024版本，于是自己去用2057.dll的英语，通过GPT-4o翻译后使用。

## 安装
将dll放到根目录即可，在option - display 选择“中文(简体， 中国)” ，重新启动软件。
建议对应版本使用，其他版本暂未可知问题，请慎用。
